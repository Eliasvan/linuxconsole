Supported hardware

Currently supported devices are:

  • PS/2 port controllers
    • i8042 AT kbd+aux controller
    • ct82c710 aux controller
    • Q40 kbd controller
    • Acorn RiscPC kbd controller
    • parport to aux adapter
  • Serial ports
    • serial port line discipline
  • Game ports
    • NS558 compatible gameports
      • Legacy ISA gameports

      • Aureal Vortex2
      • Trident 4DWave/DX
